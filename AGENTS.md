# Learnify Academy UI Rules

Tech Stack:
- React
- TailwindCSS
- React Router
- MERN Stack

Goal:
Enhance the UI/UX into a premium modern SaaS-style education platform.

Design Style:
- Modern glossy UI
- Glassmorphism
- Premium spacing
- Elegant shadows
- Smooth transitions
- Professional typography
- Clean visual hierarchy
- Startup aesthetic
- Inspired by Stripe, Linear, Framer, and Vercel

Constraints:
- Preserve all functionality
- Do not modify backend/API logic
- Do not break authentication
- Keep React Router unchanged
- Keep responsiveness
- Keep TailwindCSS only
- Avoid unnecessary libraries
- Keep code modular and maintainable

UI Preferences:
- rounded-xl or rounded-2xl
- subtle gradients
- soft shadows
- backdrop blur where appropriate
- premium hover effects
- consistent spacing system
- modern buttons and cards

Avoid:
- excessive animations
- neon colors
- overdesigned layouts
- unnecessary complexity
